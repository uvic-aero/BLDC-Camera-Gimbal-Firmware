BLDC Camera Gimbal for Uvic AERO

UVic AERO is a student led engineering organization dedicated to the design and deployment of aircraft and payloads for aeronautical reconnaissance operations and competitions. This project aims to develop a hardware and software system that would improve the aerial photography capabilites of AEROs exsiting aircraft-mounted gimbal system. This is acheived through in-flight image stabilization and aided by an object tracking system.
